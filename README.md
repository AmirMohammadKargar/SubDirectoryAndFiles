# SubDirectoryAndFiles
Simple python script to show all subdirectories and files in current directory

<h3>Run</h3>

```
python dirShow.py
```

<h3>Example</h3>

```
[am-kargar@DESKTOP-TFV96NE Downloads]$ python dirShow.py 
Directory Path: /home/am-kargar/Downloads
|__753103760_6432286c-4f59-4d4d-a3c2-7a005f1b1098.xcf
|__hromedriver_linux64.zip
|__ky.jpg
|__irShow.py
|__753103760_6432286c-4f59-4d4d-a3c2-7a005f1b1098.png
|__Telegram Desktop
  |__920x400.zip
```
